Robot Overview

Our robot, ID-10-T, was created using 4mm extrusions, Rev pieces, and 3D printed parts. It has a 4-motor drivetrain that uses mechanum wheels. There are 2 spinning wheels on the front (controlled by motors) that serves as the intake. The intake is attached to LongRobotics linear slides (controlled by motors) that allows blocks to be stacked. A continuous servo moves a hook that allows the robot to move the foundation during autonomous mode.
